# Wikiworks spellchecker

- Plugin files in `custom_plugins/wikiworksspellchecker`

## Install httpserver
```bash
npm install -g httpserver
```

## Run
```bash
httpserver
```
- Then open http://0.0.0.0:8080
- Set `Devtool > Networks > Disable cache`
